# RCISA Johannesburg North Website

Welcome to the official website of the Reformed Church International South Africa (RCISA) - Johannesburg North Branch.

## About

This website serves the Johannesburg North branch of RCISA, which is part of the greater Reformed Church in Zimbabwe (RCZ) family. The church is led by Reverend Munikwa and his wife, serving the community in Fontainebleau, Randburg.

- **Location**: ℅ Rabie Street & Fourth Ave N, Fontainebleau, Randburg, 2194, South Africa

## Setup Instructions

1. **Clone or download** this repository
2. **Replace placeholder images** in the `assets/images/` folder:
   - `rcz-logo.png` - Official RCZ logo (current SVG is a placeholder)
   - `church-building.jpg` - Photo of the church building
   - `rev-munikwa.jpg` - Photo of Reverend Munikwa
   - `mrs-munikwa.jpg` - Photo of Mrs. Munikwa

3. **Update contact information** in `index.html`:
   - Email address (currently: info@rcisa-jnorth.org)
   - Phone number (if available)
   - Social media links

4. **Customize Google Maps** embed:
   - Get the exact coordinates for your church location
   - Update the iframe src in the Contact section

5. **Open `index.html`** in a web browser to view the website

## Customization

### Colors
The website uses CSS variables for easy color customization. Edit these in `assets/css/style.css`:

```css
:root {
    --primary-color: #0d2c6c;      /* Dark blue */
    --secondary-color: #fbbf24;     /* Gold/Yellow */
    --accent-color: #3b82f6;        /* Light blue */
}
```

### Fonts
Current fonts:
- **Headings**: Playfair Display (serif)
- **Body**: Inter (sans-serif)

To change fonts, update the Google Fonts link in `index.html` and the font-family in `style.css`.

## About RCZ

The Reformed Church in Zimbabwe (RCZ) was founded on September 9, 1891, at Morgenster Mission in Masvingo province by missionaries from the Dutch Reformed Church (DRC) in South Africa. The church has grown to serve communities across Zimbabwe and South Africa through spiritual guidance, education, and healthcare.

**Motto**: "Kuda Kwenyu Ngakuitwe" (Thy Will Be Done)
